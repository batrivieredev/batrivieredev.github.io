argc et argv en C

Le argc et argv sont des paramètres utilisés dans la fonction main d'un programme en C pour gérer les arguments de la ligne de commande.

  • argc (argument count) est un entier qui représente le nombre d'arguments passés au programme, y compris le nom du programme lui-même.
  • argv (argument vector) est un tableau de chaînes de caractères (tableau de char*) qui contient les arguments passés au programme. Chaque élément de ce tableau est une chaîne de caractères représentant un argument.

Exemple d'utilisation

#include <stdio.h>

int main(int argc, char *argv[]) {
    printf("Nombre d'arguments : %d\n", argc);

    for (int i = 0; i < argc; i++) {
        printf("Argument %d : %s\n", i, argv[i]);
    }

    return 0;
}

Dans cet exemple, si vous exécutez le programme avec les arguments suivants :

./mon_programme arg1 arg2 arg3

La sortie sera :

Nombre d'arguments : 4
Argument 0 : ./mon_programme
Argument 1 : arg1
Argument 2 : arg2
Argument 3 : arg3

Conclusion

argc et argv sont des outils puissants pour gérer les arguments de la ligne de commande en C. Ils permettent aux utilisateurs de passer des informations au programme lors de son exécution, ce qui augmente la flexibilité et l'interactivité des applications.
