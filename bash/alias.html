Les alias Bash

Alias temporaires

Les alias temporaires sont valables uniquement dans la session de terminal actuelle. Pour créer un alias temporaire :

alias nom_alias='commande'

Exemples :

# Raccourci pour git status
alias gs='git status'

# Raccourci pour ls -la
alias ll='ls -la'

# Raccourci pour revenir au dossier parent
alias ..='cd ..'

Alias permanents

Pour créer des alias permanents, il faut les ajouter dans le fichier .bashrc :

  1. Ouvrir le fichier .bashrc :
    2. nano ~/.bashrc
  2. Ajouter vos alias à la fin du fichier :
    3. # Mes alias personnalisés
alias gs='git status'
alias ll='ls -la'
alias ..='cd ..'
  3. Sauvegarder et recharger le fichier .bashrc :
    4. source ~/.bashrc

Bonnes pratiques

  • Choisir des noms courts mais explicites
  • Regrouper les alias par catégorie dans le fichier .bashrc
  • Ajouter des commentaires pour expliquer les alias complexes
  • Éviter de redéfinir des commandes existantes
