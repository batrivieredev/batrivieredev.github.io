Liste des Commandes Bash

Voici une liste des commandes Bash les plus courantes, accompagnées d'une brève description et d'exemples pratiques. Ces commandes permettent d'interagir avec le système, de gérer des fichiers, de naviguer dans les répertoires, et plus encore.

Commandes de navigation

  • pwd - Affiche le répertoire de travail actuel. pwd
  • ls - Liste les fichiers et dossiers du répertoire courant. # Liste tous les fichiers ls # Liste avec détails ls -l
  • cd - Change le répertoire courant. # Aller dans le répertoire "Documents" cd Documents # Revenir au répertoire parent cd ..

Commandes de gestion de fichiers

  • touch - Crée un fichier vide. # Crée un fichier nommé fichier.txt touch fichier.txt
  • mkdir - Crée un répertoire. # Crée un dossier appelé "mon_dossier" mkdir mon_dossier
  • cp - Copie des fichiers ou des dossiers. # Copie fichier1.txt en fichier2.txt cp fichier1.txt fichier2.txt # Copie tout le contenu du dossier1 dans dossier2 cp -r dossier1/ dossier2/
  • mv - Déplace ou renomme des fichiers. # Renommer fichier.txt en nouveau_nom.txt mv fichier.txt nouveau_nom.txt # Déplacer fichier.txt vers le dossier "Documents" mv fichier.txt Documents/
  • rm - Supprime des fichiers ou des répertoires. # Supprimer fichier.txt rm fichier.txt # Supprimer un répertoire et son contenu rm -r mon_dossier/

Commandes d'affichage et de lecture de fichiers

  • cat - Affiche le contenu d'un fichier. # Affiche le contenu de fichier.txt cat fichier.txt
  • more / less - Affiche un fichier page par page. # Lire fichier.txt page par page less fichier.txt
  • head / tail - Affiche les premières ou dernières lignes d'un fichier. # Les 10 premières lignes de fichier.txt head fichier.txt # Les 5 dernières lignes tail -n 5 fichier.txt

Commandes système

  • echo - Affiche un texte ou une variable. # Affiche "Bonjour, monde !" echo "Bonjour, monde !"
  • whoami - Affiche l'utilisateur actuel. whoami
  • uname - Affiche des informations sur le système. # Informations de base sur le système uname # Détails supplémentaires uname -a
  • top - Affiche les processus en cours. top
  • kill - Termine un processus. # Termine un processus avec l'ID 1234 kill 1234

Commandes réseau

  • ping - Vérifie la connectivité réseau avec un hôte. ping google.com
  • ifconfig - Affiche les configurations réseau (requiert des privilèges root). ifconfig
  • scp - Copie des fichiers vers/depuis une machine distante. # Copie fichier.txt vers un serveur distant scp fichier.txt utilisateur@serveur:/chemin/du/dossier

Commandes Nano

  • Ctrl + G - Affiche l'aide # Utilisez Ctrl+G pour voir toutes les commandes
  • Ctrl + O - Sauvegarde le fichier # Utilisez Ctrl+O puis Enter pour sauvegarder
  • Ctrl + X - Quitte nano # Utilisez Ctrl+X pour quitter (il vous demandera de sauvegarder si nécessaire)
  • Ctrl + K - Coupe la ligne # Utilisez Ctrl+K pour couper une ligne entière
  • Ctrl + U - Colle le texte # Utilisez Ctrl+U pour coller le texte coupé
  • Ctrl + W - Recherche dans le fichier # Utilisez Ctrl+W pour rechercher du texte

Commandes Vi/Vim

  • i - Passe en mode insertion # Appuyez sur i pour commencer à écrire
  • Esc - Retourne au mode normal # Appuyez sur Echap pour quitter le mode insertion
  • :w - Sauvegarde le fichier # En mode normal, tapez :w puis Enter
  • :q - Quitte vi/vim # En mode normal, tapez :q puis Enter
  • :wq - Sauvegarde et quitte # En mode normal, tapez :wq puis Enter
  • dd - Supprime une ligne # En mode normal, tapez dd
  • yy - Copie une ligne # En mode normal, tapez yy pour copier
  • p - Colle après le curseur # En mode normal, tapez p pour coller
  • /motif - Recherche un texte # En mode normal, tapez /texte puis Enter
  • u - Annule la dernière action # En mode normal, tapez u pour annuler

Commandes Emacs

  • C-x C-f - Ouvre un fichier # Utilisez Ctrl+x puis Ctrl+f pour ouvrir un fichier
  • C-x C-s - Sauvegarde le fichier # Utilisez Ctrl+x puis Ctrl+s pour sauvegarder
  • C-x C-c - Quitte Emacs # Utilisez Ctrl+x puis Ctrl+c pour quitter
  • C-k - Coupe la ligne à partir du curseur # Utilisez Ctrl+k pour couper une ligne
  • C-y - Colle le texte coupé ou copié # Utilisez Ctrl+y pour coller
  • C-s - Recherche vers l'avant # Utilisez Ctrl+s pour rechercher du texte
  • C-r - Recherche vers l'arrière # Utilisez Ctrl+r pour rechercher en arrière
  • C-g - Annule la commande en cours # Utilisez Ctrl+g pour annuler une commande
  • M-x - Execute une commande par son nom # Utilisez Alt+x pour entrer une commande

Conclusion

Cette liste est une introduction aux commandes Bash les plus courantes. Il existe bien d'autres commandes et options que vous pouvez explorer en consultant la documentation officielle ou en utilisant man commande.
