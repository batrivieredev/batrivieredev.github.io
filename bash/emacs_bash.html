Emacs - Éditeur de texte

Emacs est un éditeur de texte puissant et extensible, souvent utilisé pour le développement logiciel.

Commandes principales

Action Commande
Ouvrir un fichier emacs fichier.txt
Enregistrer un fichier CTRL + x, CTRL + s
Quitter Emacs CTRL + x, CTRL + c
Annuler une action CTRL + /
Rechercher dans un fichier CTRL + s

Caractéristiques principales

  • Hautement configurable
  • Support de nombreux plugins et extensions
  • Environnement de développement complet
  • Support multi-langages

Emacs est particulièrement apprécié pour sa flexibilité et sa capacité à être personnalisé selon les besoins de l'utilisateur.
