Nano - Éditeur de texte

Nano est un éditeur de texte simple et facile à utiliser, parfait pour les débutants.

Commandes principales

Action Commande
Ouvrir un fichier nano fichier.txt
Enregistrer un fichier CTRL + O
Quitter Nano CTRL + X
Couper une ligne CTRL + K
Coller une ligne CTRL + U
Rechercher un mot CTRL + W

Avantages de Nano

  • Interface simple et intuitive
  • Apprentissage rapide
  • Parfait pour les modifications rapides
  • Disponible sur la plupart des systèmes Unix/Linux

Nano est idéal pour les débutants ou pour des modifications rapides de fichiers en ligne de commande.
