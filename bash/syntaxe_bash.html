Introduction à la Syntaxe Bash

Bash (Bourne Again Shell) est un interpréteur de commandes utilisé dans les systèmes d'exploitation Unix/Linux. Il permet d'exécuter des commandes, d'écrire des scripts et d'automatiser des tâches. La compréhension de la syntaxe Bash est essentielle pour exploiter pleinement les fonctionnalités d'un terminal.

1. Commandes de base

  • Afficher un message : echo "Bonjour, monde!"
  • Changer de répertoire : cd /chemin/du/dossier
  • Lister les fichiers : ls -l
  • Créer un fichier vide : touch fichier.txt

2. Variables en Bash

Les variables permettent de stocker des valeurs temporaires pour les utiliser dans des scripts.

Exemple :

# Déclaration d'une variable nom="Alice" # Utilisation de la variable echo "Bonjour, $nom!"

3. Structures conditionnelles

Bash permet d'exécuter des blocs de code conditionnellement grâce aux instructions if, elif, et else.

Exemple :

# Vérifier si un fichier existe if [ -f fichier.txt ]; then echo "Le fichier existe." else echo "Le fichier n'existe pas." fi

4. Boucles

Les boucles permettent de répéter une action plusieurs fois. Les deux principales boucles en Bash sont for et while.

Exemples :

# Boucle for for i in {1..5}; do echo "Compteur : $i" done # Boucle while compteur=1 while [ $compteur -le 5 ]; do echo "Compteur : $compteur" compteur=$((compteur + 1)) done

5. Fonctions en Bash

Les fonctions permettent de regrouper un ensemble de commandes pour les réutiliser facilement.

Exemple :

# Déclaration d'une fonction dire_bonjour() { echo "Bonjour, $1!" } # Appel de la fonction avec un argument dire_bonjour "Alice"

6. Redirections et Pipes

  • Rediriger la sortie vers un fichier : commande > fichier.txt
  • Ajouter à un fichier existant : commande >> fichier.txt
  • Utiliser un pipe pour connecter deux commandes : commande1 | commande2

Exemple :

# Trouver un mot dans un fichier et afficher uniquement les résultats correspondants cat fichier.txt | grep "mot"

Conclusion

Bash est un outil puissant pour les développeurs et les administrateurs système. En combinant les commandes, les variables, les boucles, et les redirections, vous pouvez automatiser des tâches complexes avec des scripts simples. N'hésitez pas à pratiquer et à explorer la documentation pour aller plus loin.
