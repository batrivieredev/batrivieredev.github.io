Vi/Vim - Éditeur de texte

Vi (et sa version améliorée Vim) est un éditeur très populaire dans les environnements Unix/Linux. Il est rapide, puissant et entièrement contrôlable au clavier.

Modes de Vi/Vim

Mode Description Activation
Normal Mode par défaut pour la navigation et les commandes Esc
Insertion Pour saisir du texte i, a, o, I, A, O
Visuel Pour sélectionner du texte v, V, Ctrl+v
Commande Pour entrer des commandes Ex :

Navigation

Action Commande Description
Déplacements basiques h, j, k, l Gauche, bas, haut, droite
Mots w, b, e Début mot suivant, début mot précédent, fin mot
Ligne 0, $, ^ Début ligne, fin ligne, premier caractère non-blanc
Document gg, G Début document, fin document
Écran H, M, L Haut, milieu, bas de l'écran
Défilement Ctrl+f, Ctrl+b Page suivante, page précédente

Édition

Action Commande Description
Insertion i, a, o Avant curseur, après curseur, nouvelle ligne dessous
Suppression x, dd, dw Caractère, ligne entière, mot
Copier yy, yw, y$ Ligne, mot, jusqu'à fin de ligne
Coller p, P Après curseur, avant curseur
Annuler/Refaire u, Ctrl+r Annuler dernière action, refaire
Remplacer r, R Un caractère, mode remplacement

Recherche et Remplacement

Action Commande Description
Recherche avant /motif Recherche vers l'avant
Recherche arrière ?motif Recherche vers l'arrière
Navigation résultats n, N Occurrence suivante, précédente
Remplacement :%s/ancien/nouveau/g Remplace toutes les occurrences

Commandes de Fichier

Action Commande Description
Sauvegarder :w Enregistre le fichier
Quitter :q, :q! Quitter, quitter sans sauvegarder
Sauver et quitter :wq, ZZ Enregistre et quitte
Nouveau fichier :e fichier Ouvre un nouveau fichier

Fonctionnalités Avancées

Action Commande Description
Macros q{lettre} Enregistrer une macro
Exécuter macro @{lettre} Exécuter la macro enregistrée
Marques m{lettre} Définir un marqueur
Aller à la marque '{lettre} Aller au marqueur
Split horizontal :split Divise la fenêtre horizontalement
Split vertical :vsplit Divise la fenêtre verticalement

Vi/Vim est extrêmement puissant et ces commandes ne représentent qu'une partie de ses capacités. La maîtrise de ces commandes de base permet déjà d'être très productif.
