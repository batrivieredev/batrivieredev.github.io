Fonctions
 

Fonctions en C

Les fonctions en C sont des blocs de code qui effectuent une tâche spécifique. Elles permettent de structurer le programme, de réutiliser du code et de rendre le code plus lisible. Une fonction peut prendre des entrées, appelées paramètres, et peut renvoyer une valeur.

Exemple de fonction simple

Voici un exemple d'une fonction qui additionne deux nombres, avec une explication de chaque ligne :


#include <stdio.h> // Inclusion de la bibliothèque standard pour l'entrée et la sortie.

int additionner(int a, int b) { // Déclare une fonction qui prend deux entiers en paramètres et retourne un entier.
    return a + b; // Retourne la somme des deux entiers.
}

int main() { // Début du programme principal.
    int x = 5; // Déclare une variable entière x et l'initialise à 5.
    int y = 10; // Déclare une variable entière y et l'initialise à 10.
    int resultat = additionner(x, y); // Appelle la fonction additionner avec x et y, stocke le résultat.
    printf("La somme de %d et %d est %d\\n", x, y, resultat); // Affiche le résultat formaté.
    return 0; // Indique que le programme s'est terminé correctement.
}
  • #include <stdio.h> : Permet d'utiliser les fonctions standard telles que printf
  • int additionner(int a, int b) : Déclare une fonction qui prend deux paramètres entiers et retourne un entier
  • return a + b : Calcule la somme des deux entiers et renvoie ce résultat
  • int main() : Point d'entrée du programme.
  • int x = 5; : Initialise une variable x avec la valeur 5
  • int y = 10; : Initialise une variable y avec la valeur 10
  • int resultat = additionner(x, y); : Appelle la fonction additionner avec x et y comme arguments
  • printf("La somme...") : Affiche la somme des deux nombres avec un formatage
  • return 0; : Indique que le programme s'est exécuté correctement

Exemple de fonction avec des paramètres par référence

Voici un exemple d'une fonction qui échange deux valeurs, avec une explication :

#include <stdio.h> // Inclusion de la bibliothèque standard pour l'entrée et la sortie.

void echanger(int *a, int *b) { // Déclare une fonction prenant deux pointeurs en paramètres.
    int temp = *a; // Stocke la valeur pointée par a dans temp.
    *a = *b; // Copie la valeur pointée par b dans l'adresse pointée par a.
    *b = temp; // Copie la valeur de temp dans l'adresse pointée par b.
}

int main() { // Début du programme principal.
    int x = 5; // Initialise x à 5.
    int y = 10; // Initialise y à 10.
    printf("Avant échange : x = %d, y = %d\\n", x, y); // Affiche les valeurs avant l'échange.
    echanger(&x, &y); // Appelle la fonction avec les adresses de x et y.
    printf("Après échange : x = %d, y = %d\\n", x, y); // Affiche les valeurs après l'échange.
    return 0; // Indique que le programme s'est terminé correctement.
}
  • #include <stdio.h> : Permet d'utiliser printf
  • void echanger(int *a, int *b) : Déclare une fonction qui prend deux pointeurs d'entiers
  • int temp = *a; : Stocke la valeur pointée par a dans une variable temporaire
  • *a = *b; : Copie la valeur pointée par b dans l'adresse pointée par a
  • *b = temp; : Copie la valeur temporaire dans l'adresse pointée par b
  • int main() : Point d'entrée du programme
  • printf("Avant échange...") : Affiche les valeurs avant l'appel à echanger
  • echanger(&x, &y) : Passe les adresses de x et y pour les modifier directement
  • printf("Après échange...") : Affiche les valeurs après l'échange
  • return 0; : Indique que le programme s'est terminé correctement

Conclusion

Les fonctions sont un élément fondamental de la programmation en C. Elles permettent de structurer le code, de le rendre plus lisible et de réutiliser des blocs de code. En utilisant des paramètres et des valeurs de retour, les fonctions peuvent être très flexibles et puissantes.
