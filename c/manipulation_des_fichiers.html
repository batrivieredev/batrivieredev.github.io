Manipulation des Fichiers
 

Manipulation de Fichiers en C

1. Introduction

La manipulation de fichiers en C permet de lire et d'écrire des données dans des fichiers. Cela est essentiel pour la persistance des données, permettant aux programmes de stocker et de récupérer des informations entre les exécutions.

2. Types de Fichiers

Il existe deux types de fichiers en C :

  • Fichiers texte : Contiennent des données sous forme de texte lisible.
  • Fichiers binaire : Contiennent des données sous forme binaire, non lisible directement.

3. Fonctions de Base

Les fonctions principales pour manipuler des fichiers en C incluent :

  • fopen() : Ouvre un fichier.
  • fclose() : Ferme un fichier.
  • fread() : Lit des données d'un fichier binaire.
  • fwrite() : Écrit des données dans un fichier binaire.
  • fprintf() : Écrit des données formatées dans un fichier texte.
  • fscanf() : Lit des données formatées à partir d'un fichier texte.

4. Exemple de Lecture et d'Écriture de Fichiers

Voici un exemple simple qui montre comment écrire dans un fichier texte et le lire ensuite :

Fichier : exemple_fichier.c

#include <stdio.h>

int main() {
    FILE *fichier;
    char nom[50];

    // Écriture dans un fichier
    fichier = fopen("exemple.txt", "w");
    if (fichier == NULL) {
        printf("Erreur à l'ouverture du fichier.\n");
        return 1;
    }
    fprintf(fichier, "Bonjour, ceci est un exemple de fichier.\n");
    fclose(fichier);

    // Lecture du fichier
    fichier = fopen("exemple.txt", "r");
    if (fichier == NULL) {
        printf("Erreur à l'ouverture du fichier.\n");
        return 1;
    }
    while (fgets(nom, sizeof(nom), fichier) != NULL) {
        printf("%s", nom);
    }
    fclose(fichier);

    return 0;
}

5. Compilation et Exécution

Pour compiler et exécuter ce programme, utilisez les commandes suivantes dans un terminal :

gcc exemple_fichier.c -o exemple_fichier
./exemple_fichier

Après l'exécution, un fichier nommé exemple.txt sera créé, contenant le texte écrit par le programme.

6. Conclusion

La manipulation de fichiers en C est une compétence essentielle pour le développement de logiciels. En comprenant comment ouvrir, lire, écrire et fermer des fichiers, vous pouvez créer des programmes qui interagissent avec les données de manière efficace.
