Introduction à SQLite

SQLite est une bibliothèque légère de gestion de base de données relationnelle intégrée, parfaite pour les applications locales et les prototypes.

Caractéristiques principales

  • Sans serveur : Base de données intégrée dans l'application
  • Fichier unique : Toute la base dans un seul fichier
  • Transactionnel : ACID compliant
  • Zéro configuration : Pas besoin d'installation ou setup
  • Cross-platform : Fonctionne sur tous les systèmes

Utilisation en Python

import sqlite3

# Connexion à la base
conn = sqlite3.connect('database.db')
cursor = conn.cursor()

# Création de table
cursor.execute('''
    CREATE TABLE IF NOT EXISTS users (
        id INTEGER PRIMARY KEY AUTOINCREMENT,
        name TEXT NOT NULL,
        email TEXT UNIQUE,
        created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
    )
''')

# Insertion de données
cursor.execute('''
    INSERT INTO users (name, email)
    VALUES (?, ?)
''', ('John Doe', 'john@example.com'))

# Commit et fermeture
conn.commit()
conn.close()

Opérations CRUD

Create & Read

-- Insertion
INSERT INTO users (name, email)
VALUES ('Alice', 'alice@example.com');

-- Lecture simple
SELECT * FROM users WHERE id = 1;

-- Lecture avec JOIN
SELECT u.name, p.title
FROM users u
JOIN posts p ON p.user_id = u.id
WHERE u.email LIKE '%@example.com';

Update & Delete

-- Mise à jour
UPDATE users
SET name = 'Alice Smith'
WHERE id = 1;

-- Suppression
DELETE FROM users WHERE id = 1;

-- Transaction
BEGIN TRANSACTION;
UPDATE accounts SET balance = balance - 100
WHERE id = 1;
UPDATE accounts SET balance = balance + 100
WHERE id = 2;
COMMIT;

Utilisation avec Flask

from flask import Flask, g
import sqlite3

app = Flask(__name__)

def get_db():
    if 'db' not in g:
        g.db = sqlite3.connect('database.db')
        g.db.row_factory = sqlite3.Row
    return g.db

@app.teardown_appcontext
def close_db(error):
    db = g.pop('db', None)
    if db is not None:
        db.close()

@app.route('/users')
def get_users():
    db = get_db()
    users = db.execute('SELECT * FROM users').fetchall()
    return [dict(user) for user in users]
