Configuration de Git

Configuration initiale

Avant d'utiliser Git, vous devez configurer votre identité :

git config --global user.name "Votre Nom"
git config --global user.email "votre@email.com"

Configuration de l'éditeur

Choisissez votre éditeur préféré pour les messages de commit :

# Pour VSCode
git config --global core.editor "code --wait"

# Pour Vim
git config --global core.editor "vim"

# Pour Nano
git config --global core.editor "nano"

Alias Git utiles

Les alias permettent de créer des raccourcis pour les commandes fréquentes :

# Configuration des alias
git config --global alias.co checkout
git config --global alias.br branch
git config --global alias.ci commit
git config --global alias.st status

# Alias plus avancés
git config --global alias.lg "log --graph --pretty=format:'%Cred%h%Creset -%C(yellow)%d%Creset %s %Cgreen(%cr) %C(bold blue)<%an>%Creset' --abbrev-commit"
git config --global alias.unstage "reset HEAD --"

Configuration des couleurs

Personnalisez les couleurs de Git pour une meilleure lisibilité :

git config --global color.ui true
git config --global color.status.changed "blue"
git config --global color.status.untracked "red"
git config --global color.status.added "green"
