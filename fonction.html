Fonctions en C

Les fonctions en C sont des blocs de code qui effectuent une tâche spécifique. Elles permettent de structurer le programme, de réutiliser du code et de rendre le code plus lisible. Une fonction peut prendre des entrées, appelées paramètres, et peut renvoyer une valeur.

Déclaration d'une fonction

La déclaration d'une fonction comprend plusieurs éléments :

  • Type de retour : Le type de donnée que la fonction va renvoyer (par exemple, int, float, void, etc.).
  • Nom de la fonction : Un identifiant unique pour la fonction.
  • Paramètres : Une liste d'arguments que la fonction peut accepter, encadrée par des parenthèses.

La syntaxe générale est la suivante :

type_de_retour nom_de_fonction(type_param1 param1, type_param2 param2, ...) {
    // Corps de la fonction
}

Exemple de fonction simple

Voici un exemple d'une fonction qui additionne deux nombres :

#include <stdio.h>

// Déclaration de la fonction
int additionner(int a, int b) {
    return a + b; // Retourne la somme de a et b
}

int main() {
    int x = 5;
    int y = 10;
    int resultat = additionner(x, y); // Appel de la fonction
    printf("La somme de %d et %d est %d\n", x, y, resultat);
    return 0;
}

Exemple de fonction avec des paramètres par référence

Les fonctions peuvent également modifier des variables en utilisant des pointeurs. Voici un exemple d'une fonction qui échange les valeurs de deux variables :

#include <stdio.h>

// Déclaration de la fonction
void echanger(int *a, int *b) {
    int temp = *a; // Stocke la valeur de a
    *a = *b;       // Assigne la valeur de b à a
    *b = temp;     // Assigne la valeur de temp à b
}

int main() {
    int x = 5;
    int y = 10;
    printf("Avant échange : x = %d, y = %d\n", x, y);
    echanger(&x, &y); // Appel de la fonction avec les adresses de x et y
    printf("Après échange : x = %d, y = %d\n", x, y);
    return 0;
}

Conclusion

Les fonctions sont un élément fondamental de la programmation en C. Elles permettent de structurer le code, de le rendre plus lisible et de réutiliser des blocs de code. En utilisant des paramètres et des valeurs de retour, les fonctions peuvent être très flexibles et puissantes.
