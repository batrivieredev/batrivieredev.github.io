Fonctions Variadiques
 

Fonction Variadique

#include <stdio.h>
#include <stdarg.h>

int somme(int nombre, ...) {
	int total = 0;
	va_list args;

	// Initialiser args pour le nombre d'arguments
	va_start(args, nombre);

	// Ajouter chaque argument à total
	for (int i = 0; i < nombre; i++) {
		total += va_arg(args, int);
	}

	// Nettoyer args
	va_end(args);

	return total;
}

int main() {
	printf("La somme est : %d\n", somme(4, 1, 2, 3, 4)); // Affiche : La somme est : 10
	return 0;
}

Explication du Code

Ce programme C illustre l'utilisation d'une fonction variadique, c'est-à-dire une fonction qui accepte un nombre variable d'arguments.

  • Inclusion des bibliothèques : Les bibliothèques <stdarg.h> et <stdio.h> sont utilisées. <stdarg.h> fournit les outils nécessaires pour gérer les arguments variables.
  • Fonction somme : Cette fonction accepte un premier paramètre nombre, indiquant le nombre d'arguments supplémentaires passés. Les étapes de la fonction sont :
    • Initialisation de va_list args avec va_start, qui pointe vers le premier argument variable.
    • Une boucle for qui utilise va_arg pour récupérer chaque argument et l’ajouter à total.
    • Libération de la mémoire associée avec va_end.
  • Fonction main : Elle appelle la fonction somme avec quatre arguments supplémentaires (1, 2, 3, 4) et affiche leur somme dans la console.
