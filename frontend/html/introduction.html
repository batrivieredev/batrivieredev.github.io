Introduction à HTML5

Introduction Balises Attributs Formulaires Ressources
HTML (HyperText Markup Language) est le langage standard pour créer et structurer des pages web.

Structure de base

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <title>Titre de la page</title>
</head>
<body>
    <h1>Titre principal</h1>
    <p>Contenu de la page</p>
</body>
</html>

Nouveautés HTML5

  • Balises sémantiques (<header>, <nav>, <main>)
  • Audio et vidéo natifs
  • Canvas pour les graphiques
  • Stockage local
  • Formulaires améliorés

Exemple pratique

<article>
    <header>
        <h1>Mon premier article</h1>
        <p>Publié le <time datetime="2024-03-06">6 mars 2024</time></p>
    </header>

    <main>
        <p>Le contenu de mon article...</p>

        <section>
            <h2>Une section importante</h2>
            <p>Des détails supplémentaires...</p>
        </section>
    </main>

    <footer>
        <p>Auteur: Baptiste RIVIÈRE</p>
    </footer>
</article>

Bonnes pratiques

  • Utiliser une structure sémantique
  • Valider le code HTML
  • Optimiser pour l'accessibilité
  • Maintenir une indentation cohérente
  • Commenter le code important
