Introduction à Git

Qu'est-ce que Git ?

Git est un système de contrôle de version distribué qui permet de suivre les modifications apportées aux fichiers d'un projet au fil du temps. Il permet de :

  • Garder un historique des modifications
  • Travailler en équipe sur le même code source
  • Gérer différentes versions du projet
  • Revenir à des versions précédentes si nécessaire

Concepts de base

  • Dépôt (Repository) : L'endroit où Git stocke toutes les données de votre projet
  • Commit : Un instantané de votre code à un moment donné
  • Branche : Une ligne de développement indépendante
  • Remote : Un dépôt distant (comme sur GitHub)
