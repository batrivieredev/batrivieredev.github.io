Qu'est-ce qu'un attribut HTML ?

Les attributs HTML fournissent des informations supplémentaires sur les éléments HTML. Ils sont placés dans la balise d'ouverture et sont constitués d'un nom et d'une valeur.

Exemples d'attributs courants

Voici quelques-uns des attributs les plus utilisés en HTML :

  • href : Définit l'URL d'un lien (<a>).
  • src : Définit la source d'une image (<img>).
  • alt : Texte alternatif pour une image, utile pour l'accessibilité.
  • class : Associe un élément à une ou plusieurs classes CSS.
  • id : Attribue un identifiant unique à un élément.
  • style : Applique des styles CSS directement à un élément.
  • title : Affiche une info-bulle au survol de l'élément.
  • target : Définit où ouvrir un lien (_blank pour un nouvel onglet).

Exemples de code

Utilisation de href et target

                
                <a href="https://example.com" target="_blank">Visitez ce site</a>

Utilisation de src et alt

                
                <img src="image.jpg" alt="Description de l'image">

Utilisation de id et class

                
                <div id="conteneur" class="classe1 classe2">Contenu ici</div>

Utilisation de style

                
                <p style="color: blue; font-size: 18px;">Texte en bleu</p>

Utilisation de title

                
                <p title="Ceci est une info-bulle">Passez la souris ici</p>
