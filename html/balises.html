Qu'est-ce qu'une balise HTML ?

Les balises HTML sont utilisées pour structurer et organiser le contenu d'une page web. Chaque balise est entourée de chevrons (<>) et peut contenir du texte, des images ou d'autres balises.

Exemples de balises courantes

Voici quelques-unes des balises HTML les plus utilisées :

  • <h1> à <h6> : Définit les titres (du plus important au moins important).
  • <p> : Définit un paragraphe.
  • <a href="url"> : Crée un lien hypertexte vers une autre page.
  • <img src="image.jpg" alt="Description"> : Insère une image.
  • <div> : Utilisé comme conteneur pour organiser le contenu.

Exemples de code

Utilisation des titres

                
                <h1>Titre Principal</h1>
                <h2>Sous-titre</h2>
                <h3>Sous-sous-titre</h3>

Création d'un paragraphe

                
                <p>Ceci est un paragraphe contenant du texte descriptif.</p>

Ajout d'un lien hypertexte

                
                <a href="https://example.com">Visitez ce site</a>

Insertion d'une image

                
                <img src="image.jpg" alt="Description de l'image">

Utilisation de la balise <div> pour structurer une page

                
                <div>
                    <h2>Section de contenu</h2>
                    <p>Ceci est une section de texte entourée d'une balise div.</p>
                </div>
