Introduction au HTML

Qu'est-ce que HTML ?

HTML (HyperText Markup Language) est le langage standard utilisé pour créer des pages web. Il permet de structurer le contenu d'une page en utilisant des balises. Chaque balise a une signification spécifique et est utilisée pour définir des éléments comme des titres, des paragraphes, des images, des liens, des listes, des tableaux, et bien plus encore.

Structure de base d'une page HTML

Voici un exemple de structure de base d'une page HTML :

                
                <!DOCTYPE html>
                <html lang="fr">
                <head>
                    <meta charset="UTF-8">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Ma première page</title>
                </head>
                <body>
                    <h1>Bienvenue sur ma page</h1>
                    <p>Ceci est un paragraphe.</p>
                </body>
                </html>

Cette structure est essentielle pour toute page HTML. Voici une explication de chaque élément :

  • <!DOCTYPE html> : Indique que le document est en HTML5.
  • <html lang="fr"> : Définit la langue de la page.
  • <head> : Contient les métadonnées, comme l'encodage et le titre.
  • <meta charset="UTF-8"> : Spécifie l'encodage des caractères pour assurer une bonne lecture.
  • <title> : Définit le titre affiché dans l'onglet du navigateur.
  • <body> : Contient le contenu visible de la page.

Quelques balises courantes

Voici quelques balises essentielles en HTML :

  • <h1> à <h6> : Définit les titres (du plus grand au plus petit).
  • <p> : Définit un paragraphe.
  • <a href="url"> : Crée un lien hypertexte.
  • <img src="image.jpg" alt="Description"> : Insère une image.
  • <ul> et <li> : Créent une liste non ordonnée.
  • <ol> et <li> : Créent une liste ordonnée.
