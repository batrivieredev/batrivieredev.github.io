Qu'est-ce que printf ?

La fonction printf est utilisée en C pour afficher des données sur la sortie standard (généralement l'écran). Elle fait partie de la bibliothèque standard stdio.h.

La syntaxe de base de printf est la suivante : 

printf("format", arguments);

Le format est une chaîne de caractères qui peut contenir des spécificateurs de format, et les arguments sont les valeurs à afficher.

Exemples d'utilisation

1. Afficher une chaîne de caractères

printf("Bonjour, monde !");

Ce code affichera : Bonjour, monde !

2. Afficher des entiers

int a = 10;
printf("La valeur de a est : %d", a);

Ce code affichera : La valeur de a est : 10

3. Afficher des nombres à virgule flottante

float b = 3.14;
printf("La valeur de b est : %.2f", b);

Ce code affichera : La valeur de b est : 3.14

4. Afficher plusieurs valeurs

int x = 5;
float y = 2.5;
printf("x = %d, y = %.1f", x, y);

Ce code affichera : x = 5, y = 2.5

Spécificateurs de format courants

  • %d : pour les entiers
  • %.2f : pour les nombres à virgule flottante (2 décimales)
  • %s : pour les chaînes de caractères
  • %c : pour les caractères

Conclusion

La fonction printf est un outil puissant pour afficher des informations en C. En utilisant des spécificateurs de format, vous pouvez contrôler la manière dont les données sont affichées.
