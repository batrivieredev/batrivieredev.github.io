Commandes Python avec Exemples

Python offre un large éventail de commandes et de fonctions intégrées qui facilitent le développement. Voici une liste des commandes courantes accompagnées d'exemples pour vous aider à comprendre leur utilisation.

1. Affichage avec print()

La commande print() permet d'afficher du texte ou des valeurs dans la console.

# Exemple d'utilisation print("Bonjour, monde!") # Affiche : Bonjour, monde! print(5 + 3) # Affiche : 8

2. Entrée utilisateur avec input()

La commande input() permet de lire une saisie de l'utilisateur.

# Exemple d'utilisation nom = input("Entrez votre nom : ") print(f"Bonjour, {nom}!")

3. Conversion de types

Python fournit des fonctions pour convertir des types de données.

  • int() : Convertit en entier.
  • float() : Convertit en flottant.
  • str() : Convertit en chaîne.
# Exemple d'utilisation nombre = "42" entier = int(nombre) # Convertit en entier print(entier + 1) # Affiche : 43

4. Manipulation des listes

  • append() : Ajoute un élément à la fin.
  • remove() : Supprime un élément.
  • sort() : Trie la liste.
# Exemple d'utilisation ma_liste = [3, 1, 4] ma_liste.append(2) print(ma_liste) # Affiche : [3, 1, 4, 2] ma_liste.sort() print(ma_liste) # Affiche : [1, 2, 3, 4]

5. Boucles avec for et while

Les boucles permettent de répéter des actions.

# Exemple de boucle for for i in range(5): print(f"Compteur : {i}") # Exemple de boucle while compteur = 0 while compteur < 5: print(f"Compteur : {compteur}") compteur += 1

6. Gestion des conditions avec if, elif, else

# Exemple d'utilisation age = 20 if age < 18: print("Mineur") elif age == 18: print("Juste majeur") else: print("Majeur")

7. Fonctions avec def

Les fonctions permettent de regrouper des blocs de code réutilisables.

# Exemple d'utilisation def dire_bonjour(nom): return f"Bonjour, {nom}!" print(dire_bonjour("Alice")) # Affiche : Bonjour, Alice!

8. Fichiers en Python

Vous pouvez lire et écrire dans des fichiers avec Python.

# Exemple d'écriture with open("exemple.txt", "w") as fichier: fichier.write("Hello, world!") # Exemple de lecture with open("exemple.txt", "r") as fichier: contenu = fichier.read() print(contenu) # Affiche : Hello, world!

Conclusion

Cette liste couvre les commandes Python les plus courantes. N'hésitez pas à explorer davantage et à expérimenter pour maîtriser ce langage puissant et polyvalent.
