Récursion en C

La récursion est une technique de programmation où une fonction s'appelle elle-même pour résoudre un problème. Elle est souvent utilisée pour simplifier des problèmes complexes en les décomposant en sous-problèmes plus simples. La récursion est généralement composée de deux parties :

  • Cas de base : La condition qui arrête la récursion. C'est le point où la fonction ne s'appelle plus elle-même.
  • Appel récursif : La partie où la fonction s'appelle elle-même avec des arguments modifiés.

Exemple de récursion : Factorielle

L'un des exemples les plus courants de récursion est le calcul de la factorielle d'un nombre. La factorielle d'un entier positif n (notée n!) est le produit de tous les entiers positifs jusqu'à n.

La définition mathématique de la factorielle est :

  • n! = n × (n-1)! pour n > 0
  • 0! = 1 (cas de base)

Implémentation en C

#include <stdio.h>

int factorielle(int n) {
    // Cas de base
    if (n == 0) {
        return 1;
    } else {
        // Appel récursif
        return n * factorielle(n - 1);
    }
}

int main() {
    int nombre = 5;
    printf("La factorielle de %d est %d\n", nombre, factorielle(nombre));
    return 0;
}

Exemple de récursion : Fibonacci

Un autre exemple classique est la suite de Fibonacci, où chaque nombre est la somme des deux précédents :

  • F(0) = 0
  • F(1) = 1
  • F(n) = F(n-1) + F(n-2) pour n > 1

Implémentation en C

#include <stdio.h>

int fibonacci(int n) {
    // Cas de base
    if (n == 0) {
        return 0;
    } else if (n == 1) {
        return 1;
    } else {
        // Appel récursif
        return fibonacci(n - 1) + fibonacci(n - 2);
    }
}

int main() {
    int nombre = 6;
    printf("Le terme %d de la suite de Fibonacci est %d\n", nombre, fibonacci(nombre));
    return 0;
}

Conclusion

La récursion est un outil puissant qui permet de résoudre des problèmes de manière élégante et concise. Cependant, il est important de s'assurer qu'il existe un cas de base pour éviter des appels récursifs infinis, ce qui pourrait entraîner un dépassement de pile.
