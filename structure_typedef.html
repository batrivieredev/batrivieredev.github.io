Structures et typedef en C

Structures en C

Une structure (ou struct) est un type de données composite qui permet de regrouper plusieurs variables de différents types sous un même nom. Cela est particulièrement utile lorsque vous devez représenter un objet ou un concept qui a plusieurs attributs.

Déclaration d'une structure

struct Point {
    int x;
    int y;
};

Dans cet exemple, nous avons défini une structure nommée Point qui contient deux membres : x et y, tous deux de type int.

Utilisation d'une structure

struct Point p1;
p1.x = 10;
p1.y = 20;

printf("Point p1: (%d, %d)\n", p1.x, p1.y);

Typedef en C

Le mot-clé typedef permet de créer un alias pour un type de données, ce qui peut rendre le code plus lisible et plus facile à gérer. Cela est particulièrement utile pour les structures, car cela vous permet de ne pas avoir à écrire le mot-clé struct chaque fois que vous déclarez une variable de ce type.

Déclaration d'un typedef pour une structure

typedef struct {
    int x;
    int y;
} Point;

Dans cet exemple, nous avons créé un alias Point pour la structure, ce qui nous permet de l'utiliser sans le préfixe struct.

Utilisation d'un typedef

Maintenant, vous pouvez créer des variables de type Point directement :

Point p2;
p2.x = 30;
p2.y = 40;

printf("Point p2: (%d, %d)\n", p2.x, p2.y);

Exemple complet

Voici un exemple complet qui combine tout cela :

#include <stdio.h>

// Définition de la structure avec typedef
typedef struct {
    int x;
    int y;
} Point;

int main() {
    // Création d'une instance de Point
    Point p1;
    p1.x = 10;
    p1.y = 20;

    Point p2;
    p2.x = 30;
    p2.y = 40;

    // Affichage des points
    printf("Point p1: (%d, %d)\n", p1.x, p1.y);
    printf("Point p2: (%d, %d)\n", p2.x, p2.y);

    return 0;
}

Conclusion

Les structures et le typedef sont des outils puissants en C qui vous permettent de créer des types de données complexes et de rendre votre code plus lisible. En utilisant des structures, vous pouvez regrouper des données connexes, et avec typedef, vous simplifiez leur utilisation dans votre code.
