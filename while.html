Comprendre les Boucles while en C

Qu'est-ce qu'une boucle while ?

Une boucle while en C est une structure de contrôle qui permet d'exécuter un bloc de code tant qu'une condition spécifiée est vraie. Elle est souvent utilisée pour répéter des actions jusqu'à ce qu'une certaine condition soit remplie.

La syntaxe de base d'une boucle while est la suivante : 

while (condition) {
    // bloc de code à exécuter
}

Exemples d'utilisation

1. Compter de 1 à 5

int i = 1;
while (i <= 5) {
    printf("%d\n", i);
    i++;
}

Ce code affichera :

1
2
3
4
5

2. Somme des nombres de 1 à n

int n = 5;
int somme = 0;
int i = 1;

while (i <= n) {
    somme += i;
    i++;
}
printf("La somme des nombres de 1 à %d est : %d", n, somme);

Ce code affichera : La somme des nombres de 1 à 5 est : 15

3. Boucle infinie

int j = 0;
while (1) { // condition toujours vraie
    printf("Cette boucle ne s'arrêtera jamais !\n");
    j++;
    if (j >= 5) break; // pour éviter une boucle infinie dans cet exemple
}

Ce code affichera : Cette boucle ne s'arrêtera jamais ! jusqu'à ce que la condition de sortie soit atteinte.

Conclusion

Les boucles while sont très utiles pour exécuter des blocs de code de manière répétée tant qu'une condition est vraie. Elles sont souvent utilisées pour des tâches telles que le traitement de données, la validation d'entrées, et bien plus encore.
